# EmissionsMatching
Used to match emissions factors based on active date range

After package installation Create custom setting record for emissions 
Matching, Set update and insert triggers as desired, Logging and batch 
size if required. Set 
matching field to desired date ( StartDate should be default )

Next add Recalculate emission checkbox to vehicle, Stationary and waste 
energy use pages layouts. 

Add recalculate emissions, Master emissions set and Date active fields to 
Refridgerant, Electricity, Waste and Other emissions sets pagelayouts. 

Verify matching is working correctly with a test record & running apex 
classes.
